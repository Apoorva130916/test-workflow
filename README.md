# 🚀 GitHub Pages Automation Demo

This repository is used to demonstrate the automation of GitHub Pages setup using AutomationEdge.

## 📌 Use Case
Automatically create and configure a GitHub Pages site using GitHub REST API.

## ⚙️ Workflow
- API call to enable GitHub Pages
- Uses main branch as source
- Deploys static content automatically

## 🌐 Expected Output
After successful execution, the site will be available at:

https://Apoorva130916.github.io/test-workflow

## 🛠️ Tools Used
- AutomationEdge Process Studio
- Advanced REST Client (ARC)
- GitHub REST API
